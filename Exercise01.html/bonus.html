HTML & CSS Quick-Refrence Guide

Here we will learn some HTML tags and CSS properties

HTML Tags

  • Heading Tag

This tag is used to create titles or headings.
<H1> is for big headings, <H6> is for small text 

Example:-<H1>This for big heading</H1>

  • Paragraph Tag

This tag defines a paragraph

Example:-<p>Html stands for hyper text markup language</p>

  • Italic Text Tag

This tag makes the text italic

Example:-<i>She loves Chocolates</i>

  • Bold Text Tag

This tag makes the text bold

Example:-<b>She loves dancing</b>

  • Line Break Tag

This tag is to create a break in the line of text

Example:-line<br>break

  • Code Tag

This tag displays a code.

Example:-<code>Hello World</code>

  • Ordered Lists Tag

This tag creates a lists with numbers.

Example:-
            <ol>
            <li>Sara</li>
            <li>Ahmed</li>
            </ol>

  • Unordered Lists Tag

This tag creates a lists with bullet points.

Example:-
            <ul>
            <li>Cats</li>
            <li>Dogs</li>
            </ul>

  • Image Tag

This tag displays a image.

Example:-<>

  • Anchor tag

This tag is used to create hyperlink.

Example:-<a href="https://www.google.com">Link to google</a>

  • Audio tag

This tag is used to add sound content on website.

Example:-<audio controls src="Sound Effect.mp3"></audio>

  • Video Tag

This tag is used to add video on webiste.

Example:-<video controls src="Video.mp3"></video>

  • Form Tag

This tag is used to create a form for user input

Example:-
          <form>
          <label for=>"123"Email </label>
          <input id="123" type="text" placeholder="Email">
          </form>

  • Div Tag

This tag is used to organize components in webpage 

Example:-<div><h5>About me</h5><p>i am 18 years old</p></div>

  • Article Tag

This tag is used to represent independent piece of content 

Example:-<article><h4>Article Title</h4><p>---</p></article>

  • Footer Tag

This tag is used to represent the bottom of a webpage 

Example:-<footer>My Website</footer>

CSS Properties

  • Color Text

This changes the font color

p {color: #5c2a47;}

  • Background Color

This changes the background color

body {background-color:rgb(252, 255, 223)}

  • Font Family

Changes the text of the font

body {font-family: "Times new Roman" ,serif;}

  • Font Size

Changes the size of the font

p {font-size: 1em}

  • Text Align

Changes the position of the text

h1 {text-align:left }

  • Border

sets a border around a text

h1 {
    border: 2px solid rgb(0, 0, 0);
    border-radius: 5px;
    display:inline-block;
    padding:5px;
      }

  • Margin

Margins are used to create space around a elements border

h1 {
        margin: 25px 50px 10px ;
      }

  • Padding

This creates an extra space within an element

div {
        padding: 3x 3px;
      }
